Detailed Description

template<typename T>
class ml::ArenaSTLAllocator< T >

STL-compatible allocator using ArenaAllocator.

Template Parameters
TThe type of objects to allocate.

This class provides an STL-compatible allocator that uses ArenaAllocator for memory management. It can be used with standard containers like std::vector, std::list, and std::deque to allocate memory from an arena, improving performance and memory locality.

See also
ArenaAllocator for the underlying allocation strategy.
