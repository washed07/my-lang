Detailed Description

An allocator using arena allocation strategy.

Mnages memory in large chunks to provide fast allocation and deallocation of small to medium-sized objects, minimizing fragmentation and overhead. It is particularly well-suited for compiler use cases where many temporary objects are created.

See also
ArenaStats for tracking allocation statistics.
ArenaChunk for individual memory chunk management.
