Detailed Description

A container for statistics about ArenaAllocator usage.

Holds various statistics about the memory usage of an ArenaAllocator instance, including:

  • Total allocated bytes,
  • Requested bytes,
  • Allocation count,
  • Chunk count,
  • Peak usage,
  • Current usage,
  • Wasted bytes due to fragmentation or alignment. It also provides methods to compute fragmentation ratio and efficiency.
    See also
    ArenaAllocator for usage context.
