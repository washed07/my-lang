Detailed Description

Represents a single memory chunk in a ArenaAllocator.

Encapsulates a contiguous block of memory managed by the ArenaAllocator. It tracks the size of the chunk, the amount of memory used, and provides methods for allocation within the chunk.

See also
ArenaAllocator for context.
